### Set 자료구조 

* 중복을 허용하지 않는 자료구조이다. 

```
Set<String> set = new HashSet<>();

set.add("Carol");
set.add("Alice");
set.add("Dave");
set.remove("Carol");
set.add("Erin");
set.add("Erin");

set.size();
set.contains("carol")

remove.add.contains.size. 

```

* HashSet 
해시 테이블을 기반으로 하였다. 
null 저장 가능.
* TreeSet
Binary Search Tree 를 기반으로 한 것이다.
null 저장 불가.
